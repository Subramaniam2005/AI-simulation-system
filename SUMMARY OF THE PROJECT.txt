# 🧠 PROJECT TITLE

## **Multi-Agent AI Group Chat Simulation System**

A memory-driven multi-agent AI system where multiple AI personalities dynamically participate in a group discussion using:

* runtime memory retrieval
* graph database storage
* LLM-generated responses

---

# 🎯 PROJECT OBJECTIVE

The goal of the project is to simulate a realistic AI-driven group conversation between multiple personalities discussing a fixed topic:

```text id="jlwm40"
Weekend Planning
```

Each AI agent:

* has its own personality memory
* retrieves context dynamically
* generates behavior-specific responses

The system architecture is designed to later support:

* dynamic agent insertion
* scalable memory retrieval
* persistent conversational memory

---
